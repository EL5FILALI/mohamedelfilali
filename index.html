Mode d’emploi de Méthodo Prépas

De Novice à Légende des Concours – Guide pour Professeurs et Étudiants

1. Présentation générale

Méthodo Prépas est une plateforme interactive accessible à l’adresse methodoprepas.web.app. Elle aide les étudiants à se préparer aux concours scientifiques et commerciaux grâce à des exercices de dissertation, résumé et synthèse, tout en offrant un système de gamification motivant (points, badges, progression) pour les étudiants, et des outils de gestion pour les professeurs.

Pourquoi l’utiliser ? Méthodo Prépas rend l’apprentissage ludique, structuré et adapté aux besoins des classes préparatoires, tout en facilitant la supervision pour les professeurs !

2. Connexion et inscription

Pour les étudiants

Accès initial : Rendez-vous sur le site, choisissez "Étudiant" sur l’écran d’accueil.

Inscription/Connexion avec un compte : - Remplissez Email et Mot de passe. - Cliquez sur "Inscription" ou "Connexion". - Option : Continuez sans compte via "Continuer sans compte" (progrès non sauvegardés).

Un compte permet de sauvegarder vos points, badges et niveaux pour une progression continue !

Pour les professeurs

Accès initial : Choisissez "Professeur", puis connectez-vous ou inscrivez-vous avec Email et Mot de passe.

Rôle : Gérer les contenus, superviser les étudiants via l’espace "La Chambre des Sages".

Profitez de Méthodo Prépas pour enrichir vos cours et motiver vos étudiants avec des défis interactifs !

3. Navigation pour les étudiants

Après connexion, accédez à la page d’accueil avec :

  • Barre de gamification : Votres titres (Novice, Penseur, etc.), points, badges, et une barre de progression.
  • Choix des voies : Sélectionnez "Voie Scientifique" ou "Voie Commerciale".

3.1. Voie Scientifique

Dissertation scientifique : 9 niveaux (Citation → Parties III & Conclusion). Résumé scientifique : 5 niveaux (Thèse → Résumé).

3.2. Voie Commerciale

Dissertation commerciale : Similaire, avec contenus adaptés. Résumé commercial : Idem. Synthèse commerciale : 5 niveaux (Problématique → Perfection).

3.3. Fonctionnement des défis

- Niveaux verrouillés jusqu’à validation du précédent. - Niveau 1 : Cliquez "Valider". - Niveaux suivants : Rédigez dans la zone texte, cliquez "Vérifier". - Feedback : Messages détaillés avec points et conseils d’amélioration. - Nouveau Défi : Cliquez "Nouveau Défi" pour un contenu aléatoire.

3.4. Gamification

- Points : Gagnés pour bonnes réponses, perdus (-5) après 3 erreurs. - Titres : Débloqués par paliers (Novice, Apprenant, etc.). - Badges : Récompensent les niveaux clés (ex. : Analyste Éclairé au niveau 2).

Motivez vos étudiants avec ce système ludique et engageant – idéal pour les classes préparatoires !

4. Navigation pour les professeurs

Accédez à "La Chambre des Sages" pour gérer les contenus :

  • Sélectionnez une section : Via le menu déroulant.
  • Gérez les contenus : Blocs affichant Professeur, Date, Contenu, et Actions (Modifier/Supprimer pour vos contenus, ou Supprimer pour contenus orphelins).
  • Ajoutez un contenu : Remplissez le formulaire, cliquez "Ajouter".
  • Modifiez un contenu : Cliquez "Modifier", ajustez, puis "Enregistrer".
  • Supprimez un contenu : Cliquez "Supprimer", confirmez.
  • Contenus orphelins : Identifiés en rouge, supprimables manuellement ou via "Nettoyer les contenus orphelins".
Facilitez votre enseignement en créant des défis personnalisés et en maintenant une base de données propre avec Méthodo Prépas !

5. Conseils et bonnes pratiques

Pour les étudiants : Lisez les instructions, utilisez les feedbacks pour progresser, et engagez-vous dans la gamification pour rester motivés.

Pour les professeurs : Ajoutez des contenus variés, surveillez les orphelins, et encouragez vos étudiants avec les badges et titres.

Transformez vos cours en une aventure éducative captivante avec Méthodo Prépas – adoptez-le dès aujourd’hui !

6. Dépannage

- Problème avec `sci-diss` : Vérifiez les contenus dans Firebase (`sci-diss/citations`). Reconnectez-vous pour recharger les données initiales. - Contenus orphelins : Utilisez le nettoyage pour les supprimer. - Erreur technique : Ouvrez la console JavaScript (F12) et contactez [support@methodoprepas.com].
